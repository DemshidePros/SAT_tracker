# SAT Performance Analytics Tracker

A web-based application built to track and analyze daily SAT preparation progress, built with Python, Streamlit, Pandas, and Altair.

## Features
- **Attempt Logger:** Log practice questions by topic and subtopic with success tracking.
- **Dynamic Stats:** Automatically compute and sort performance statistics by success percentage.
- **Interactive Analytics:** Filter and visualize daily success rates over time using an interactive chart with a fixed 0–100% scale.

## Tech Stack
- **Frontend / UI:** Streamlit
- **Data Processing:** Pandas
- **Visualization:** Altair
- **Storage:** JSON

## How to Run Locally
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py
